# 📊 Customer Churn Prediction

This project predicts whether a customer will churn using Machine Learning.

## 🚀 Features
- Logistic Regression model
- Data preprocessing and scaling
- Streamlit web app
- Probability-based predictions

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Streamlit

## 📈 Model Performance
- Accuracy: ~73%
- Recall (Churn): ~80%
